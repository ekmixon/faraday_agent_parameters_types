============
Installation
============


Stable release
--------------

To install faraday_agent_parameters_types, run this command in your terminal:

.. code-block:: console

    $ pip install faraday_agent_parameters_types

This is the preferred method to install faraday_agent_parameters_types, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for faraday_agent_parameters_types can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/bmoyano/faraday_agent_parameters_types

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/bmoyano/faraday_agent_parameters_types/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/bmoyano/faraday_agent_parameters_types
.. _tarball: https://github.com/bmoyano/faraday_agent_parameters_types/tarball/master
